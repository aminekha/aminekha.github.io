Project Demo

This is a question answering web app built using Tensorflow 2.0 and Django. The app uses the T5 pretrained model which comes in 3 sizes (small, medium, and large).

This project comes with a beautiful easy to use UI for beginners instead of working with the Terminal. You only need to follow the instruction in the github repo and you will have a ready-to-use web app to answer your questions exactly like Google Assistant or Apple's Siri.
